# Proyecto Vía Segura
Proyecto Final: Nuñez, Vanesa - Ramírez, Tatiana DWN4AV

Es una app ciudadana colaborativa que les permite a los usuarios moverse por la ciudad con mayor seguridad, 
ofreciendo información en tiempo real sobre el estado de las calles y promoviendo la participación activa de la
comunidad en la construcción de entornos urbanos más seguros.

- - - - - - - - - - - - 
La app permite: 
* Crear reportes urbanos: ingresando la dirección en el mapa, podés crear un reporte nuevo o sumarte a un reclamo ya existente para aumentar la visibilidad y evitar duplicados. 
* Ver todos los reportes y filtrar según categorías, estado de resolución, más recientes, etc.
* Contactos de confianza: para informarlos en caso de emergencia y recorrido.
* Recorrido seguro: ingresando dirección de destino podés ver la mejor ruta a ese lugar y compartir tu trayecto en tiempo real
con tu contacto de confianza.
* Chat general: para enterarte de las novedades locales e interactuar con la comunidad. Más rápido que reportar un incidente y mejor comunicación entre los ciudadanos. 
* Chat privado: para hablar con una persona puntual. 
* Información útil: permite encontrar puntos de interés cercanos, ingresando una dirección y radio de búsqueda. Se pueden encontrar datos útiles de hospitales, comisarías, bomberos y farmacias. 
* Botón de emergencia: activa el botón antipánico para conectar con 911.
* Perfil: en esta sección podés ver y editar tus datos personales, así como también los reportes que realizaste hasta el momento.

- - - - - - - - 
El proyecto está realizado con vue.js y supabase como base de datos. Para iniciarlo:
* npm install (para instalar dependencias)
* npm run dev (abrir servidor local)
